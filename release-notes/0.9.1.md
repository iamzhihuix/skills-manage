# skills-manage 0.9.1

## English

Small maintenance release focused on full-path display consistency across platforms, with a bit of README polish.

### Highlights

- Show full absolute paths across Central, Platform, Settings, Global Search, and platform-edit flows
- Render Windows paths with drive letters and backslashes in display-oriented UI
- Add a `Star History` section to the English and Chinese READMEs

### Packages

- `skills-manage_0.9.1_windows_x64.msi`
- `skills-manage_0.9.1_windows_x64.zip`
- `skills-manage_0.9.1_macos_universal.dmg`
- `skills-manage_0.9.1_macos_universal.zip`
- `skills-manage_0.9.1_macos_universal.tar.gz`

### Notes

- This is a maintenance follow-up to `0.9.0`, centered on UI path presentation rather than new platform capability.
- Desktop builds remain unsigned, and macOS notarization is still deferred.
- Checksums should be generated from the final release artifacts before publishing or mirroring builds.

### Changelog

- Replaced `~`-collapsed path display with full absolute paths in the main path-oriented UI surfaces
- Normalized Windows path display to full drive-letter + backslash formatting
- Updated custom platform auto-generated directory defaults to match the detected platform home style
- Added `Star History` to both README files
- Extended tests to lock in the new path display rules
