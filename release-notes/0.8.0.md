# skills-manage 0.8.0

## English

First public release of `skills-manage`.

### Highlights

- Unified desktop app for managing AI agent skills across built-in and custom platforms
- Central skills library, per-platform installs, collections, and full skill detail previews
- Discover scanner for project-level skill libraries
- Marketplace browsing, AI explanations, and GitHub repository import with token support and mirror fallback
- Command palette, bilingual UI, and Catppuccin theming
- Search and large-list performance improvements for central, discover, and global search flows

- Launch the Tauri desktop app and core platform/central skill management flows
